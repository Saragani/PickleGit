# Keyboard Shortcuts

Every shortcut below can be rebound in Settings → Shortcuts. These are the defaults:

| Action | Shortcut |
|---|---|
| Refresh | F5 |
| Open Repository | Ctrl+O |
| Clone Repository | Ctrl+Shift+N |
| Close Tab | Ctrl+W |
| Next Tab | Ctrl+Tab |
| Previous Tab | Ctrl+Shift+Tab |
| Push | Ctrl+Shift+P |
| Pull | Ctrl+Shift+L |
| Fetch | Ctrl+Shift+F |
| Create Branch | Ctrl+B |
| Next Hunk (in diff view) | F7 |
| Previous Hunk (in diff view) | Shift+F7 |
| Command Palette | Ctrl+P |
| Find Commits | Ctrl+F |
| Focus Commit List | Ctrl+1 |

The **Command Palette** (`Ctrl+P`) is worth learning early — it lets you search for and run any
action by name, which is often faster than remembering a specific key combination or hunting
through toolbars.
